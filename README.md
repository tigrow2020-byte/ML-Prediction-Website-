# 🔮 ML Marks Prediction Website

A machine learning web app that predicts student marks based on study hours.

## 🛠️ Tech Stack
- Python, Scikit-learn, Flask, HTML

## ✨ Features
- Enter study hours
- Predicts expected marks instantly
- Linear Regression model

## 🚀 How to Run
```bash
pip install flask scikit-learn numpy
python app.py
```
Open browser → `http://localhost:5000`

## 📂 Project Structure
```
ml_website/
├── app.py
├── model.py
└── templates/
    └── index.html
```

## 📈 Example
```
Input  : 7 hours
Output : Predicted Marks → 82.5
```
